# Horizon AI – Full Stack

**All services in one Railway project – completely free.**

## Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/yourusername/horizon-ai.git
cd horizon-ai

# 2. Run setup (clones TTS & Render API)
chmod +x setup.sh
./setup.sh

# 3. Edit .env with your API keys
nano .env

# 4. Start locally (for testing)
npm start
